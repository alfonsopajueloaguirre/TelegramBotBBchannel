# TelegramBotBBchannel
 A Telegram bot programmed in Python 3.8 with basic commands for a personal group.
 
## Comandos 
 * UNO - ¿Cómo que UNO?
 * mimir - a la cama
 * lasemilla - invoca a Jose
 * 27 - el numero mágico
 * amigos - ver códigos de amigos
 * teparto - revienta un otaku culiao
 * arcueid - la mejor
